🏊 Landing Page para Limpeza de Piscinas

Template de Landing Page profissional para empresas de limpeza e manutenção de piscinas, desenvolvido com HTML, CSS e JavaScript puro.

O objetivo deste projeto é criar um modelo reutilizável que pode ser adaptado rapidamente para diferentes prestadores de serviços de limpeza de piscinas.

Ideal para desenvolvedores que desejam criar sites rápidos para negócios locais.

🌐 Demonstração

⚙️ Como usar este template

Faça download ou clone o repositório

git clone https://github.com/desenvolvimentoweb1983/landing-page-piscina.git

Abra o projeto no editor de código

Recomendado: Visual Studio Code

Edite o arquivo script.js

Altere as informações do cliente:

nome da empresa

telefone

endereço

serviços oferecidos

tipos de piscinas atendidas

fotos da galeria

Substitua as imagens da pasta img pelas fotos reais do cliente.

📱 Contato via WhatsApp

O botão de WhatsApp utiliza a seguinte estrutura:

https://wa.me/5511999999999

Basta alterar o número no arquivo script.js.

Também é possível configurar uma mensagem automática que será enviada quando o cliente clicar no botão.

📍 Localização

O mapa utiliza Google Maps Embed.

Você pode alterar a localização substituindo o link dentro de:

mapa:"https://www.google.com/maps?q=LATITUDE,LONGITUDE&output=embed"

Exemplo:

mapa:"https://www.google.com/maps?q=-22.7046106,-46.9924357&output=embed"
